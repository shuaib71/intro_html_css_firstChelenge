HTML & CSS Basic Summary

HTML Summary

HMTL (HyperText Markup Language) is used to define our page content, structure and meaning. You don't use it for stlyling purpose. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content
  • HTML elements typically have an opening tag, content and then a cloasing tag
  • You can also have void (empty) elements like images
  • You can also configure elements with attributes
  • There's a long list of availablee elements but you gain experience over time, no worries

Lern more about all available HTML elements on MDN Reference Documentation!

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content.

  • Styles are assigned via property-value pairs
  • You can assign styles via the "style" attribute
  • To avoid code duplication, you typically use global styles (e.g. via the "stye" element) instead
  • Alternatively, you can work with exeternal stylesheet files which you "link" to
  • when working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood.

Lern more about all available CSS properties on MDN Reference Documentation!
